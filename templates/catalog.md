# Layout catalog

This is a reference for AI agents generating section variants. Each section type lists common layout patterns. Use these as starting points — adapt to the user's design system, tokens, and UI libraries.

## Hero

| Pattern | Description |
|---------|-------------|
| Centered headline | Large heading + subtext + CTA buttons, all centered. Clean and universal. |
| Split text + image | Two columns: headline/body/CTAs on one side, image/screenshot on the other. |
| Full-bleed dark | Dark background (or gradient/image), centered text, single CTA. High-impact. |
| Dashboard preview | Centered headline + CTAs, with a large app screenshot or mockup below that bleeds to the edge. |
| Video background | Full-width video or animated background with overlaid text and CTA. |
| Minimal one-liner | Very short headline, no subtext, single CTA. Maximum whitespace. |
| Announcement bar + hero | Small top banner (new feature/launch), then a standard centered or split hero below. |

## Logos / Social proof strip

| Pattern | Description |
|---------|-------------|
| Simple row | "Trusted by" label + logos in a horizontal row. |
| Grid cards | Logos inside bordered cards in a 3- or 6-column grid. |
| Dark strip | Full-width dark bar with muted logo marks. |
| Scrolling marquee | Infinite horizontal scroll of logos. Good for many brands. |
| With metric | "Trusted by 10,000+ teams" + logo row. Combines stats and social proof. |

## Features

| Pattern | Description |
|---------|-------------|
| Icon grid (3- or 4-col) | Grid of feature cards, each with icon, title, and short description. |
| Alternating rows | Feature highlighted on one side, image/illustration on the other; alternates per row. |
| Bento grid | Asymmetric grid (one large card spanning rows + smaller cards). Modern, editorial feel. |
| Centered list | Vertically stacked features, left-aligned with icons, inside a narrow centered column. |
| Tabs / toggle | Multiple feature groups behind tabs or a toggle; only one group visible at a time. |
| Large feature spotlight | One feature dominates the section with a big image and detailed description. |

## Pricing

| Pattern | Description |
|---------|-------------|
| Three-tier cards | Side-by-side cards (Free / Pro / Enterprise), middle highlighted. |
| Two-tier split | Two plans only, side by side, one dark one light. |
| Comparison table | Table with features as rows, plans as columns, checkmarks. |
| Toggle monthly/annual | Any card layout + a toggle that switches displayed prices. |
| Single plan + enterprise CTA | One self-serve plan card + a "Contact us" block for enterprise. |

## Testimonials

| Pattern | Description |
|---------|-------------|
| Three-card grid | Equal-sized quote cards with avatar, name, role. |
| Large centered quote | One big blockquote, centered, with attribution below. |
| Two-column highlight | One large dark quote card + two smaller light cards stacked beside it. |
| Carousel / slider | One testimonial at a time with prev/next controls. |
| Wall of love | Masonry grid of many short testimonials. High social proof density. |

## CTA (Call to action)

| Pattern | Description |
|---------|-------------|
| Centered dark banner | Dark background, centered heading + subtext + buttons. |
| Split with image | Two columns: text/CTA on one side, image on the other, inside a card. |
| Minimal inline | Heading left, button right, single row. Compact. |
| Gradient banner | Vibrant gradient background with white text and CTA. |

## FAQ

| Pattern | Description |
|---------|-------------|
| Centered accordion | Vertically stacked Q&A pairs, narrow centered column, expandable. |
| Two-column grid | Questions and answers in a 2-column grid layout. |
| Side-by-side | Heading + contact link on the left, Q&A list on the right. |

## Stats / Metrics

| Pattern | Description |
|---------|-------------|
| Four-stat row | Large numbers in a single row, centered. |
| Dark background cards | Stat cards on a dark background. |
| Inline with dividers | Stats separated by vertical dividers in a single line. |
| Counter animation | Numbers that count up when scrolled into view. |

## Footer

| Pattern | Description |
|---------|-------------|
| Four-column with logo | Logo/description left, link columns right, copyright bar at bottom. |
| Minimal centered | Horizontal link row + copyright, all centered. |
| Dark full-width | Same as four-column but with dark background. |
| Fat footer | Multiple columns, newsletter signup, social links, language switcher. |

## Newsletter / Email capture

| Pattern | Description |
|---------|-------------|
| Centered with input | Heading + description + email input + button, all centered. |
| Dark banner inline | Dark rounded card with heading left, input + button right. |
| Floating card | Card that appears overlaid or offset from the main content. |

## Team

| Pattern | Description |
|---------|-------------|
| Grid with avatars | Avatar circles in a grid with name and role below. |
| Card grid with bios | Cards with avatar, name, role, and a short bio paragraph. |
| Horizontal scroll | Team members in a horizontal scrollable row. |
